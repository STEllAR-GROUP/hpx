AMR Support LibrariesÂ¶

Some of the HPX Adaptive Mesh Refinment (AMR) applications written by Matthew Anderson can optionally use third-party libraries for multiple-precision mathematics (GNU MPFR Library, GNU Multi-Precision Library) and I/O (RNPL). While these libraries are not required, the AMR code may perform worse without the math libraries, and you will not get SDF output without the RNPL library.
